Problem 1

Define $f \colon [0,1] \to \R$ by $f(x) := x^3$ and let $P := \{ 0, 0.1, 0.4, 1 \}$. Compute \[ L(P,f)\quad \text{and} \quad U(P,f). \]

Solution: We have that: $x_0= 0, \;x_1=0.1,\; x_2=0.4, \;x_3=1.$

Let's calculate $m_i$ and $M_i$. First, recall that \[ m_i=\inf\left\{f(x): x_{i-1}\leq x\leq x_{i} \right\}. \] Then

$m_1=\inf\left\{f(x): 0\leq x\leq 0.1 \right\} = 0$
 $\quad \quad \;m_2=\inf\left\{f(x): 0.1\leq x\leq 0.4 \right\} = 0.001$
 $\quad \;\;m_3=\inf\left\{f(x): 0.4\leq x\leq 1 \right\} = 0.064$

Problem 1

Solution (cont.): On the other hand, we have that \[ M_i=\sup\left\{f(x): x_{i-1}\leq x\leq x_{i} \right\}. \]

Then

$M_1=\sup\left\{f(x): 0\leq x\leq 0.1 \right\} = 0.001$
 $\;\;\,M_2=\sup\left\{f(x): 0.1\leq x\leq 0.4 \right\} = 0.064$
 $M_3=\sup\left\{f(x): 0.4\leq x\leq 1 \right\} = 1\quad\;\,$

Problem 1

Solution (cont.): Thus we have

$L(P,f ) = m_1(x_1-x_0)+m_2(x_2-x_1)+m_3(x_3-x_2) = \displaystyle\sum_{i=1}^{3}m_i\Delta x_i$
 $=0(0.1-0)+0.001(0.4-0.1)+0.064(1-0.4)$

 $=0.001(0.3)+0.064(0.6)$ $=0.0003+0.0384\quad\,$

 $=0.0387\qquad\qquad\qquad\qquad\qquad\qquad\qquad\,$

Problem 1

Solution (cont.): And

$U(P,f ) = M_1(x_1-x_0)+M_2(x_2-x_1)+M_3(x_3-x_2) = \displaystyle\sum_{i=1}^{3}M_i\Delta x_i$
 $=0.001(0.1-0)+0.064(0.4-0.1)+(1-0.4)\;\;\,$

 $=0.001(0.1)+0.064(0.3)+0.6\qquad\qquad\qquad\;\;$

 $=0.0001+0.0192+0.6$ $=0.6193\qquad\qquad\quad\;\;$
