Contour integral

Consider \(\C\) a contour \(z(t)\) \(t\in [a,b]\) \(z_1=z(a)\) \(z_2=z(b)\); and \(f\) is pwc on \(C\).

Properties:

\((*)\) Linearity

  • \(\alpha \in \C:\) \(\displaystyle \int_C \left(\alpha f \right) \left(z\right) dz = \alpha \int_C f \left(z\right) dz \).
  • \(\displaystyle\int_C \left( f + g\right) \left(z\right) dz = \int_C f \left(z\right) dz + \int_C g \left(z\right) dz\).
  • \(C_1+ C_2\) defines a conoutr when end point of \(C_1\) is equal to start point of \(C_2\).

Contour integral

Properties (cont):

\((*)\) Given a contour \(C\), we define \(-C\) as follows: \[ w(t) = z(-t), \quad -b\leq t \leq -a. \]

Then (check with change of parameter formula): \[ \int_{-C} f\left(z\right) dz = - \int_{C} f\left(z\right) dz. \]

Hence \(C_1 - C_2\) is defined when end point of \(C_1\) is equal to start point of \(-C_2\), which is equal to end point of \(C_2\).

Contour integral

Example: Evaluate   \(I=\displaystyle \int_C \conj{z}\, dz\)   where \[C: z=2e^{i\theta}, -\pi/2\leq \theta \leq \pi /2\]

\(C\) is pwc with reference to this parametrisation (indeed differentiable) and \(f\) is continuous on \(C\).

Contour integral

Example (cont)

Note that \(z'(\theta) = 2 i e^{i\theta}\). Then

\(I = \displaystyle \int_{-\pi/2}^{\pi/2} f\left( z\left(\theta \right)\right) z'\left(\theta \right) d\theta\)

\(\;\;= \displaystyle \int_{-\pi/2}^{\pi/2} \conj{\left( 2 e^{i\theta}\right)} \cdot 2i e^{i\theta} d\theta\)

\(\;\;= 4i \displaystyle \int_{-\pi/2}^{\pi/2} e^{-i\theta} e^{i\theta} d\theta\) \(=4\pi i\).

Contour integral

On \(C\), \(z\conj{z} = 4\). That is \(\conj{z} = \dfrac{4}{z}\) Then \[ \int_C \frac{dz}{z}= \pi i. \]

Anti-differentiation

Let \(D\) be a domain in \(\C\) (i.e. an open, connected subset of \(\C\)).

An anti-derivative of \(f\) on \(D\) is \(F\) such that \[F'(z) = f(z)\text{ on } D.\]
