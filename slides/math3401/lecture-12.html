Show: If \(\Omega_1,\Omega_2 \subseteq \C\) are open, then so is \(\Omega_1\cap \Omega_2\).

Proof: If \(\Omega_1\cap \Omega_2 \neq \emptyset\) done (\(\emptyset\) is open).

Since \(\Omega_1,\Omega_2\) are open, set \(\epsilon = \min \{\epsilon_1, \epsilon_2\}\) and note that \(\epsilon>0\).

Thus

\(B_{\epsilon}(z)\subset \Omega_1\) by ‚ô£Ô∏è,

Hence \(\Omega_1\cap\Omega_2\) is open

\(*\) A Point \(z\in \C\) is called an accumulation point of a set \(\Omega \subseteq \C\) if every deleted neighbourood of \(z\) intersects \(\Omega\).

Example 1: \(\Omega=\left\{\dfrac{i}{2^n}\right\}_{n\in \N}\).

Limits

Let \(f\) be a \(\C\)-valued function defined on a deleted neighbourhood of \(z\in\C\).

\(\displaystyle \lim_{z\ra z_0}f(z)=w_0\) says: Given \(\epsilon > 0\) exists \(\delta > 0\) such that \[0<|z-z_0|<\delta \implies |f(z)-w_0|<\epsilon.\]

Note: \(f\) does not have to be defined at \(z_0\).

Limits

Examples:

\( f(z) = \left\{ \begin{array}{ll} 0 & z\neq 0,\\ 1337 & z=0. \end{array} \right. \)    \(\displaystyle\lim_{z\ra 0}f(z)=0\);

\[\displaystyle \lim_{z\ra 0}\dfrac{\sin z}{z}=1.\]
