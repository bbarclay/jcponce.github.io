Taylor series in \(\C\)

Example 4: Find Maclaurin series for \(f(z) = \dfrac{1}{2+ 4z}\) on \(\C\setminus \{1/2\}\).

Note that

\(\displaystyle \frac{1}{2+4z} = \frac{1/2}{1+2z}\) \(\displaystyle =\frac{1/2}{1-(-2z)}\)

\(\qquad \quad= \displaystyle\frac{1}{2} \sum_{n=0}^{\infty} (-2z)^{n}\) for \(|-2z|\lt 1\)

\(\qquad \quad= \displaystyle\frac{1}{2} \sum_{n=0}^{\infty} (-1)^{n}2^{n-1}z^n\) for \(|z|\lt 1/2\).

Taylor series in \(\C\)

Example 5:   \(f(z) = \dfrac{1+2z^2}{z^3+z^5}\) \( = \displaystyle \frac{1}{z^3} \left( \frac{2+2z^2}{1+z^2} - \frac{1}{1+z^2} \right)\)

\(\qquad \qquad\;\; \quad = \displaystyle \frac{1}{z^3} \left( 2 - \frac{1}{1+z^2} \right)\) analytic on \(\C\setminus\{\pm i\}\).

For \(|z|\lt 1\), \(\displaystyle \frac{1}{1+z^2} = \sum_{n=0}^{\infty} (-1)^nz^{2n}\). So for \(0\lt |z|\lt 1\),

\(f(z) = \displaystyle \frac{1}{z^3} \left( 2- \left( 1-z^2+z^4-\cdots \right) \right)\)

\( = \displaystyle \frac{1}{z^3} + \frac{1}{z} -z + z^3 - \cdots\)
