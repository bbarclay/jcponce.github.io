Thomae's function

In 1875, the German mathematician Carl Johannes Thomae (1840-1921) published the book Einleitung In Die Theorie Der Bestimmten Integrale where he presented a very simple but provocative example of a continuous function on irrational numbers and discontinuous on rational numbers.

Thomae's function

Definition

Define $f \colon [0,1] \to \R$ by \begin{equation*} f(x) := \begin{cases} \dfrac{1}{q} & \text{if } x=\dfrac{p}{q} \text{ where } p,q \in \N \text{ and } p \text{ and } q \text{ have no common divisors,} \\ 0 & \text{if } x \text{ is irrational.} \end{cases} \end{equation*} Show $\int_0^1 f = 0$.
