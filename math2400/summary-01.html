Field Axioms

Definition: A set $F$ is called a field if it has two operations defined on it, addition $x+y$ and multiplication $xy$, and if it satisfies the following axioms

(A1) If $x \in F$ and $y \in F$, then $x+y \in F$.

(A2) (commutativity of addition) $x+y = y+x$ for all $x,y \in F$.

(A3) (associativity of addition)} $(x+y)+z = x+(y+z)$ for all $x,y,z \in F$.

(A4) There exists an element $0 \in F$ such that $0+x = x$ for all $x \in F$.

(A5) For every element $x\in F$, there exists an element $-x \in F$ such that $x + (-x) = 0$.

Field Axioms

(M1) If $x \in F$ and $y \in F$, then $xy \in F$.

(M2) (commutativity of multiplication) $xy = yx$ for all $x,y \in F$.

(M3) (associativity of multiplication) $(xy)z = x(yz)$ for all $x,y,z \in F$.

(M4) There exists an element $1 \in F$ (and $1 \not= 0$) such that $1x = x$ for all $x \in F$.

(M5) For every $x\in F$ such that $x \not= 0$ there exists an element $\frac{1}{x} \in F$ such that $x\left(\frac{1}{x}\right) = 1$.

(D) (distributive law) $x(y+z) = xy+xz$ for all $x,y,z \in F$.

Ordered fields

Definition: A field $F$ is said to be an ordered field if $F$ together with a relation $\lt$ has the following properties:

  1. (trichotomy) For all $x, y \in S$, exactly one of $x < y$, $x=y$, or $y < x$ holds.
  2. (transitivity) If $x,y,z \in S$ are such that $x < y$ and $y < z$, then $x < z$.
  3. For $x,y,z \in F$, $x < y$ implies $x+z < y+z$.
  4. For $x,y \in F$, $x > 0$ and $y > 0$ implies $xy > 0$.

The Set of Real Numbers

Theorem: There exists a unique ordered field $\R$ with the least-upper-bound property such that $\Q \subset \R$.

Archimedean property

Theorem:

  1. (Archimedean property) If $x, y \in \R$ and $x \gt 0$, then there exists an $n \in \N$ such that \begin{equation*} nx \gt y . \end{equation*}
  2. ($\Q$ is dense in $\R$) If $x, y \in \R$ and $x \lt y$, then there exists an $r \in \Q$ such that $x \lt r \lt y$.

Absolute value

The absolute value is defined as \begin{equation*} \abs{x} := \begin{cases} x & \text{if } x \geq 0, \\ -x & \text{if } x \lt 0 . \end{cases} \end{equation*}

Triangle inequality: $\abs{x+y} \leq \abs{x}+\abs{y}$ for all $x, y \in \R$.
