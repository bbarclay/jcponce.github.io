Example 2: \(z\mapsto \dfrac{1}{z}\) on \(\C_*\).

Define \(\xi (z) = \dfrac{z}{|z|^2}\) on \(\C_*\); \(\;\;\eta(\xi)=\conj\xi\) on \(\C\).

\(\eta \circ \xi (z) = \eta \left( \xi \left( z \right) \right) = \conj{\left( \dfrac{z}{|z|^2} \right)} = \dfrac{\conj z }{ |z|^2 } \)

\(\;\;\,\quad\quad = \dfrac{\conj z}{z \conj z} = \dfrac{1}{z}\). \(\qquad z\in \C_*\)

Basic idea with new mappings

\(\xi\) is called inversion (w.r.t the unit circle),

\(\eta\) is reflexion in the real axis.

Basic idea with new mappings

For \(w=1/z = \conj z /|z|^2\), we have

\(x+i\,y \mapsto u+i\,v\) and \( w = \dfrac{x-i\,y}{x^2+y^2} \), so \[ u = \dfrac{x}{x^2+y^2} \; (1)\quad \text{ & } \quad v = \dfrac{-y}{x^2+y^2} \; (2) \]

We can use this to show that \(1/z\) maps circles and lines in the \(z\)-plane to circles and lines in the \(w\)-plane.

A note on mapping circles and lines

Circles and lines in the \(z\)-plane can be represented as \[ A(x^2+y^2)+Bx+Cy +D = 0 \;\;(*) \]

with \(A,B, C, D\in R\), and \(B^2+C^2>4AD\).

If \(A = 0\) then we have a line. If \(A \neq 0\) then we have a circle.

A note on mapping circles and lines

From \((*)\) we have \[ \left( x+\frac{B}{2A} \right)^2 + \left( y+\frac{C}{2A} \right)^2 = \left( \frac{\sqrt{B^2+C^2-4AD}}{2A} \right)^2 . \]

Note that for \(w=1/z\), expressions (1) & (2) imply \[ D(u^2+v^2)+Bu-Cv+A=0. \]

Terminology reminder

\(*\; f: \Omega \rightarrow \C\) is 1-1 or injective say \[f(z) = f(\xi)\implies z=\xi.\]

\(*\; f: \Omega \rightarrow \Lambda\subseteq \C\) is onto or surjective says given \(\eta \in \Lambda\) exists (at least one) \(z\in \Omega: f(z)=\eta\).

Split up into:

Basic idea with new mappings

Example 1: Linear transformation

(i) \(z\mapsto Az\): Write \(A=ae^{i\alpha}\) with \(\alpha\in \R, a \in \R_+\).

So \(re^{i\theta} \mapsto \left(a r\right)e^{i\left(\theta + \alpha \right)}\) geometrically dilates (expands or contracts) modulus by a factor of \(a=|A|\), and

rotates through an angle of \(\alpha=\arg A\).

Basic idea with new mappings

Example 1: Linear transformation

(ii) \(z\mapsto z + b\): Write \(b=b_1+b_2 i\) with \(b_1,b_2\in \R\).

Then translate \(b_1\) to the right and \(b_2\) up.

\(z \mapsto A z + b\): compose (i) and (ii).

Note: domain is \(\C\).
