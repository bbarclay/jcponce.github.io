Dada una topología $\tau$ para $X$ decimos que $\mathcal{B}\subset \tau$ es una base para $\tau$ si todo abierto $A\in \tau$ puede expresarse como unión de elementos de $\mathcal{B}$. La intención de definir una base para una topología es el de hacer posible una descripción de la topología sin dar de manera explícita todos los abiertos que la comprenden.

Observemos que $\mathcal{B}=\{ X \}$ es una base para la topología trivial. Por otro lado, la colección $\mathcal{B}=\{\{x\}|\; x\in X\}$ es base para la topología discreta.

The Discrete Fourier Transform is defined as \begin{aligned} X_{k}&={\dfrac{1}{N}\sum _{n=0}^{N-1}x_{n}\cdot e^{-i k {\frac {2\pi }{N}}n}}\\ &={ \dfrac{1}{N}\sum _{n=0}^{N-1}x_{n}\left[\cos \left(k{\frac {2\pi }{N}}n\right)-i\, \sin \left({k\frac {2\pi }{N}}n\right)\right],} \end{aligned} where the last expression follows from the first one by Euler's formula. Essentially, it transforms a sequence of $N$ complex numbers $$\displaystyle \mathbf {x} :=x_{0},x_{1},\ldots ,x_{N-1}$$ into another sequence of complex numbers, $$\displaystyle \mathbf {X} :=X_{0},X_{1},\ldots ,X_{N-1}.$$


Teorema: Sean $X$ conjunto y $\mathcal{B}\subset \mathcal{P}(X)$. Existe una topología $\tau_{\mathcal{B}}$ para la cual $\mathcal{B}$ es base sí, y sólo se cumple:

  1. $X=\cup \{B\,|\, B\in \mathcal{B}\}$
  2. Dados $A,B\in \mathcal{B}$ y $x\in A\cap B$ existe $C\in \mathcal{B}$ tal que $x\in C\subset A\cap B$.
