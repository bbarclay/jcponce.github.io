Case Study 30: Snowshoe hares and Canadian lynx

Program specifications: Develop a Python program that uses Eulerâ€™s method with a step size of 0.01 years to model the populations of lynx and hares.



Program 12.2: Lotka-Volterra model of hares and lynx

# Uses Euler's method and Lotka-Volterra equations to model 
# populations of lynx and hare from 1900 to 1920.
from pylab import *

# Initialise variables for Euler's method.
stepsize = 0.01
times = arange(0 , 20+stepsize , stepsize )
a=0.484
b=0.028
c=1
d=0.032
hares = zeros(int(size(times))) 
lynx= zeros(int(size(times))) 
hares[0] = 30.0
lynx[0] = 4.0


Program 12.2: Lotka-Volterra model of hares and lynx

# Step through Euler' s method
for i in range(0,size(times)-1):
    dh = a * hares[i] - b * hares[i] * lynx[i]
    dl = - c * lynx[i] + d * hares[i] * lynx[i] 
    hares[i+1] = hares[i] + stepsize * dh 
    lynx[i+1] = lynx[i] + stepsize * dl

# Output graphs 
plot(times, hares, 'b-', linewidth=2, label='Hares') 
plot(times , lynx, 'r-', linewidth=2, label='Lynx') 
xlabel ("Time ( years since 1900)")
ylabel("Number of individuals (thousands)")
xlim (0 ,20)
ylim (0 ,70)
grid(True)
legend()
show()
