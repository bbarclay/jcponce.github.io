Limits at \(\infty\)

Example 1: Show \(\displaystyle \lim_{z\ra -1}\frac{iz+3}{z+1}= \infty.\)

Let \(f(z)=\displaystyle \frac{iz+3}{z+1}\). We want to show \(\displaystyle \lim_{z\ra -1}\frac{1}{f\left(\frac{1}{z}\right)}= 0.\)

LHS \( =\displaystyle \lim_{z\ra -1} \left(\frac{z+1}{iz+3}\right) \) \(= \frac{\displaystyle \lim_{z\ra -1} (z+1) }{\displaystyle \lim_{z\ra -1} (iz+3) } \) \(= \displaystyle \frac{0}{3-i} = 0 \) \(=\) RHS.

(via Lect 12, Theorem 2, part 4)

Limits at \(\infty\)

Example 2: Calculate \(\displaystyle \lim_{z\ra \infty}\frac{2z^3-1}{z^2-1}.\) Let \(g(z)=\displaystyle \frac{2z^3-1}{z^2-1}\).

Claim \(\displaystyle \lim_{z\ra \infty}g(z)= \infty.\) So we want to show \(\displaystyle \lim_{z\ra 0}\frac{1}{g\left(\frac{1}{z}\right)} = 0.\)

\(\displaystyle \frac{1}{g\left(\frac{1}{z}\right)} \) \(=\displaystyle \frac{1}{\frac{2\left(\frac{1}{z}\right)^3-1}{\left(\frac{1}{z}\right)^2-1}} \) \(= \displaystyle \frac{\frac{1}{z^2}-1}{\frac{2}{z^3}-1} \) \(= \displaystyle \frac{z-z^3}{2-z^3} \) \(\ra 0\) as \(z\ra \infty\).

Continuity

Let \(f\) be a \(\C\)-valued function defined on a deleted neighbourhood of \(z_0\in\C\).

\(f\) is continuos at \(z_0\)means: \(\displaystyle\lim_{z\ra z_0 } f(z) = f(z_0).\)

i. e. given \(\epsilon >0\) exists \(\delta >0\) such that \[|z-z_0|<\delta \implies |f(z)-f(z_0)|<\epsilon.\]

Continuity

Basic examples

1. If \(f: \Omega \ra U\) and \(g:U\ra W\) are continuous, so is \(g \circ f : \Omega \ra W\) (\(g\) compose with \(f\): \(\left(g\circ f\right)(z) = g\left(f(z)\right)\))

2. If \(f\) is continuous and nonzero at \(z_0\) then exists \(\epsilon>0\) such that \(f(z)\neq 0\) on \(B_{\epsilon}(z_)\). Proof on tutes.

3. \(f: x+iy \mapsto u(x,y) + v(x,y)\) is continuous if and only if \(u\) and \(v\) are continuous.

4. Obvious analogues of Theorems 1 & 2 from Lecture 12.

Differentiability in \(\R\)

\(f: \Omega \subseteq \R \ra \R \), \[ \lim_{h\ra 0 } \frac{f(x+h)-f(z)}{h}. \]

If this exists, defines \(f'(x)\).

This limit is in \(\R\).

Differentiability in \(\C\)

\(f: \Omega \subseteq \C \ra \C \), \[ \lim_{\xi\ra 0 } \frac{f(z_0+\xi)-f(z_0)}{\xi}. \qquad (1) \]

If this exists, defines \(f'(z_0)\). Write \(\Delta z\) for \(\xi\).

(1) implies \(f'(z_0) = \displaystyle \lim_{\Delta z \ra 0} \frac{f(z_0+\Delta z)- f(z_0)}{\Delta z}.\quad (2)\)

Write \(w = f(z)\): So (2) implies \(f'(z_0) = \displaystyle \lim_{\Delta z \ra 0} \frac{\Delta w}{\Delta z} = \frac{dw}{dz}(z_0).\quad (3)\)

Note (1)-(3) are equivalent.
